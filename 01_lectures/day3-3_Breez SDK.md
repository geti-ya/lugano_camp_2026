# 1. 발표자와 주제

- **발표자:** 대니 (Danny / 비트코인 결제 솔루션 기업 'Breez'의 마케팅 담당자)
- **주제:** 복잡한 비트코인 기술을 숨기고 누구나 쉽게 사용할 수 있는 결제 환경 만들기 (Breez SDK와 사용자 경험)

<br>

# 2. 용어 사전

1. **비트코인의 진짜 쓸모는 '전송'에 있다:** 비트코인을 단순히 가격이 오르길 기다리는 투자용 자산으로만 보지 말고, 인터넷으로 카카오톡 메시지를 보내듯 전 세계 어디로든 즉시 돈(가치)을 보낼 수 있는 수단으로 활용해야 한다.
2. **개발자를 위한 쉬운 도구 상자를 제공해야한다:** 비트코인 결제 시스템은 만들기가 매우 까다롭지만, 누구나 쉽게 결제 기능을 앱에 넣을 수 있도록 돕는 무료 개발 도구(Breez SDK)를 만들어 배포하고 있다.
3. **가장 중요한 것은 '압도적으로 쉬운 사용법(UX)'이다:** 지갑을 만들 때 복잡한 12개의 영단어 비밀번호를 적어야 하는 등의 불편함을 없애고, 일반 금융 앱이나 토스, 카카오페이처럼 지문 인식만으로도 곧바로 결제할 수 있을 만큼 사용법이 쉬워야 한다.

<br>

# 3. 이 세션에서 언급된 용어 목록 + 한 줄 설명

- **Breez SDK (브리즈 SDK):** 개발자들이 비트코인 결제 기능을 자신의 앱이나 웹사이트에 쉽게 조립해서 넣을 수 있도록 만들어 둔 '소프트웨어 개발 도구 상자'.
- **라이트닝 네트워크 (Lightning Network):** 느리고 비싼 비트코인의 전송 속도를 매우 빠르고 저렴하게 만들어주는 보조 기술로, 강연자는 이를 여러 결제 기술을 하나로 묶어주는 **'공용어(Common language)'**라고 불렀다.
- **노드 (Node):** 비트코인 결제를 처리하기 위해 켜두어야 하는 컴퓨터 프로그램. 예전에는 스마트폰에서 직접 이걸 돌려야 해서 불편했지만, 지금은 이 과정을 뒤로 숨겨서 사용자는 신경 쓰지 않아도 되게 개선되었다.
- **시드리스(Seedless) / 패스키(Passkey):** 비트코인 지갑을 쓸 때 꼭 종이에 적어둬야 했던 복잡한 복구 단어들(시드 구문)을 없애고, 스마트폰의 얼굴 인식이나 지문(패스키)만으로 안전하게 로그인하는 기능.
- **리퀴드(Liquid) / 스파크(Spark):** 브리즈(Breez)가 결제를 더욱 빠르고 매끄럽게 만들기 위해 활용하고 있는 최신 비트코인 보조 네트워크 기술들의 이름.

<br>

# 4. 발표자가 특히 강조한 포인트나 인상적인 발언

- **"저의 목표는 올해 80세가 되시는 저희 할머니 '마거릿(Margaret)'도 비트코인을 사용할 수 있게 만드는 것입니다."**
    - 발표자는 이를 일명 '마거릿 할머니 테스트'라고 부르며, 기술의 복잡성을 완전히 숨기고 평범한 사람들도 직관적으로 쓸 수 있는 '사용자 경험(UX)'이 무엇보다 중요하다고 강조했다.
- **"우리는 전 세계 어디로든 즉시 정보를 보낼 수 있는 놀라운 시대에 살고 있는데, 왜 '가치(돈)'는 그렇게 보낼 수 없을까요?"**
    - 비트코인이 단순히 가격만 오르락내리락하는 투자 자산에 머물러서는 안 되며, 인터넷 뉴스, 소셜 미디어, 이메일 등 우리의 모든 온라인 생활 속에서 창작자에게 바로 돈을 보낼 수 있는 '국경 없는 가치 이동 수단'이 되어야 한다고 역설했다.
