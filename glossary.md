# 용어 모음

모르는 용어가 나왔을 때 여기서 먼저 확인하세요.
최대한 쉬운 말로 풀어쓰는 것을 목표로 합니다.

---

| 용어                         | 쉬운 설명                                    |
| -------------------------- | ---------------------------------------- |
| Bitcoin                    | 중앙 기관 없이 작동하는 디지털 화폐                     |
| Lightning Network          | 비트코인 거래를 빠르고 저렴하게 처리하는 레이어. 카드 결제처럼 즉시 됨 |
| Liquid                     | 비트코인 위에 얹은 또 다른 레이어. 주로 기업 간 거래에 사용      |
| RGB                        | 비트코인 위에서 스마트컨트랙트와 토큰을 구현하는 프로토콜          |
| Ark                        | 비트코인의 프라이버시와 확장성을 높이기 위한 레이어             |
| e-cash                     | 디지털 현금. 누가 얼마를 갖고 있는지 알 수 없게 하는 기술       |
| Formal Verification (정형증명) | 코드가 수학적으로 올바른지 증명하는 것. 버그를 원천 차단하는 방법    |
| Hackathon                  | 짧은 시간 안에 팀으로 아이디어를 구현해보는 대회              |
| Hands-on                   | 직접 실습하는 방식의 교육                           |
